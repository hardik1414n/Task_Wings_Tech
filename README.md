## For Install

1. Clone The Project
    ```sh
    git clone https://github.com/hardik1414n/Task_Wings_Tech.git
    ```

2. Composer Install
    ```sh
    composer install
    ```

3. Create .env using .env-example

4. Generate New App Key
    ```sh
    php artisan key:generate 
    ```

5. Migrate Database
    ```sh
    php artisan migrate --seed
    ```
